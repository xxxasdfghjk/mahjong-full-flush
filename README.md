# 麻雀清一待ちあて

-   https://xxxasdfghjk.github.io/mahjong-full-flush/

### 待ちの選択

-   表示されているキーをクリックするか数値をキーボード入力して選択

### 回答

-   ENTER ボタンを押すか ENTER キーを入力

### 回答結果

-   回答があっていたかどうか判定される
    -   赤 : 待ちとして存在するが未答
    -   黄 : 待ちとして存在しないが回答
    -   緑 : 正答
-   すべて正答だった場合得点が加算
-   ENTER キーを入力するか ENTER ボタンを押して次の局へ

### スコア

-   高目で上がった場合の得点が加算
-   誤答の場合スコアなし

### ゲーム進行

-   東一局から南 4 局まで進行
-   親番時に得点すると連荘
    -   誤答すると流局

### 終了時

-   回答した手牌とその待ち・役の一覧を閲覧可能
-   ENTER キー入力または CONTINUE ボタンのクリックでゲームを再プレイ可能

### その他ゲーム仕様

-   面前かつリーチなしでのツモを想定
-   ドラ、赤ドラなし
-   右上の牌をクリックすると牌種を変更可能
